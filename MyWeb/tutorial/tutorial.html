Como Criar um Servidor em Python

Este tutorial mostra como criar um servidor web simples usando Python, tanto no Linux quanto no Windows.

Servidor simples com Python no Linux

1. Abra o terminal e navegue até a pasta onde estão seus arquivos HTML.

2. Execute o comando para iniciar o servidor na porta 8000:

python3 -m http.server 8000

3. Agora abra o navegador e acesse: http://localhost:8000 para ver seus arquivos servidos.

Servidor simples com Python no Windows

1. Abra o Prompt de Comando (cmd) ou o PowerShell e navegue até a pasta do seu site.

2. Execute o comando:

python -m http.server 8000

3. Abra seu navegador e acesse http://localhost:8000 para visualizar a página.

Dicas extras

  • Use a porta que preferir, substituindo o 8000 no comando.
  • Se quiser que outras máquinas da rede acessem seu servidor, execute o comando com o parâmetro:
    • python3 -m http.server 8000 --bind 0.0.0.0
  • Para parar o servidor, pressione Ctrl + C no terminal.
